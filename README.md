# LRPhoton

## Overview

LRPhoton is a Python application for SAXS/WAXS data processing and visualization dedicated to laboratory and synchrotron scattering data, including XENOCS laboratory SAXS systems and ESRF beamlines ID13 and ID02.

## Installation

### Install Python

Download the latest version on: https://www.python.org/downloads

### Install modules

In Command Prompt, type:

python -m pip install PySide6 numpy matplotlib h5py fabio requests hdf5plugin

### Install LRPhoton

Download the repository as a ZIP file from GitHub and extract it on your computer.
You can do this by clicking: Code → Download ZIP

## Quick start

1. Open the `LRPhoton` folder
2. Double-click `main.py`
3. Click **Run**

## Updates

At startup, the program automatically checks whether more recent versions of the application files are available online.
